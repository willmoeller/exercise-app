Week 2, Day 2

Muscle group(s): Shoulders & Triceps

Here are a few things to keep in mind for this workout:

  1. All percentages, where listed, indicate the percent of your one rep max. Read more about a one rep max here.
  2. Target rest times between sets and supersets are between 2 and 3 minutes.

Exercise 1

Barbell Overhead Press (strict, no legs)

(7 sets)

  1. Set 1:10 reps @ ~30% max
  2. Set 2:8 reps @ increase weight
  3. Set 3:6 reps @ increase weight
  4. Set 4:4 reps @ heaviest weight
  5. Set 5:6 reps @ decrease weight
  6. Set 6:8 reps @ decrease weight
  7. Set 7:10 reps @ ~30% max

Exercise 2

Close Grip Bench Press

(6 sets)

  1. Set 1, 2:10 reps
  2. Set 3, 4:8 reps
  3. Set 5, 6:5 reps

Exercise 3

Dumbbell Side Lateral

(5 sets)

  1. Set 1, 2:10 reps
  2. Set 3, 4:15 reps
  3. Set 5:20 reps

Exercise 4

Dumbbell Skullcrusher

(3 sets)

  1. Set 1, 2, 3:10 reps

Exercise 5

Cable or Brand Pushdown

(4 sets)

  1. Set 1, 2, 3, 4:25 reps
