Week 1 Day 5

Muscle group(s): Legs & Chest

Here are a few things to keep in mind for this workout:

  1. All percentages, where listed, indicate the percent of your one rep max. Read more about a one rep max here.
  2. Target rest times between sets and supersets are between 2 and 3 minutes.

Exercise 1

Back Squat

(4 sets)

  1. Set 1:5 reps @ 60%
  2. Set 2:5 reps @ 65%
  3. Set 3:5 reps @ 70%
  4. Set 4:5 reps @ 70%

Exercise 2

Bench Press

(9 sets)

  1. Set 1:6 reps @ 62%
  2. Set 2:6 reps @ 65%
  3. Set 3:6 reps @ 67%
  4. Set 4:8 reps @ 60%
  5. Set 5:8 reps @ 62%
  6. Set 6:8 reps @ 65%
  7. Set 7:10 reps @ 60%
  8. Set 8:10 reps @ 60%
  9. Set 9:10 reps @ 60%

Exercise 3

Bench Dips (weighted if possible)

(4 sets)

  1. Sets 1-4:10 reps

Dumbbell Rear Lateral
