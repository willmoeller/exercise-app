Brick House, Week 4 Day 2

Warm up with your favorite dynamic exercises.

All percentages indicate off of your 1 rep max. Target rest periods of ~2 minutes

Exercise #1: Front Squat (3 sets)

  • Set 1: 5 reps @ 50% of back squat 1 rep max
  • Set 2: 5 reps @ 50% of back squat 1 rep max
  • Set 3: 5 reps @ 50% of back squat 1 rep max

Exercise #2: 1 Arm DB Row (4 sets)

  • Set 1: 10 reps
  • Set 2: 5 reps
  • Set 3: 5 reps
  • Set 4: 5 reps

Exercise #3: Standing BB Overhead Press (3 sets)

  • Set 1: 10 reps @ 30-32% of bench 1 rep max
  • Set 2: 10 reps @ 30-32% of bench 1 rep max
  • Set 3: 10 reps @ 30-32% of bench 1 rep max

Exercise #4: EZ Bar or Straight Bar Skull Crushers, DB Hammer Curl super set (3 sets)

  • Set 1: 10 reps Skull Crushers
  • Super set 1: 20 reps DB Hammer Curls
  • Set 2: 10 reps Skull Crushers
  • Super set 2: 20 reps DB Hammer Curls
  • Set 3: 10 reps Skull Crushers
  • Super set 3: 20 reps DB Hammer Curls

Cardio: BB Shrugs (3 sets)

  • Set 1: 10 reps
  • Set 2: 10 reps
  • Set 3: 20 reps
  • Set 4: 20 reps
