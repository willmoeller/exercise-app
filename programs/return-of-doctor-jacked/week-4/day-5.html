Week 4, Day 5

Muscle group(s): Chest & Triceps

Here are a few things to keep in mind for this workout:

  1. All percentages, where listed, indicate the percent of your one rep max. Read more about a one rep max here.
  2. Target rest times between sets and supersets are between 2 and 3 minutes.

(12 sets)

  2. Set 2:2 reps @ 70%
  3. Set 3:2 reps @ 80%
  4. Set 4:2 reps @ 85%
  5. Set 5:3 reps @ 60%
  6. Set 6:3 reps @ 70%
  7. Set 7:3 reps @ 80%
  8. Set 8:3 reps @ 85%
  9. Set 9:4 reps @ 60%
  10. Set 10:4 reps @ 70%
  11. Set 11:4 reps @ 80%
  12. Set 12:4 reps @ 85%
