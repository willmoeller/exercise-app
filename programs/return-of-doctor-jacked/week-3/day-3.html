Week 3, Day 3

Muscle group(s): Legs

Here are a few things to keep in mind for this workout:

  1. All percentages, where listed, indicate the percent of your one rep max. Read more about a one rep max here.
  2. Target rest times between sets and supersets are between 2 and 3 minutes.

Exercise 1

Back Squat

(7 sets)

  1. Set 1:8 reps @ 60%
  2. Set 2:6 reps @ 70%
  3. Set 3:4 reps @ 80%
  4. Set 4:2 reps @ 90%
  5. Set 5:4 reps @ 80%
  6. Set 6:6 reps @ 75%
  7. Set 7:8 reps @ 65%

Exercise 2

Dumbbell RDL

(3 sets)

  1. Sets 1, 2, 3:10 reps

Exercise 3

Barbell Shrug

(4 sets)

  1. Sets 1, 2, 3, 4:10 reps

Exercise 4
