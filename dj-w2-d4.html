Week 2, Day 4

Muscle group(s): Chest & Triceps

Here are a few things to keep in mind for this workout:

  1. All percentages, where listed, indicate the percent of your one rep max.
  2. Target rest times between sets and supersets are between 2 and 3 minutes.

Exercise 1

Bench Press

(10 sets)

  1. Set 1:5 reps @ 50%
  2. Set 2:5 reps @ 60%
  3. Set 3:3 reps @ 70%
  4. Set 4:4 reps @ 70%
  5. Set 5:5 reps @ 70%
  6. Set 6:6 reps @ 70%
  7. Set 7:2 reps @ 80%
  8. Set 8:3 reps @ 80%
  9. Set 9:4 reps @ 80%
  10. Set 10:5 reps @ 80%
