Brick House, Week 4 Day 1

Warm up with your favorite dynamic exercises.

All percentages indicate off of your 1 rep max. Target rest periods of ~2.5 minutes

Exercise #1: Snatch Grip Deadlift (3 sets)

  • Set 1: 5 reps @ 50% of 1 rep max
  • Set 2: 5 reps @ 52.5% of 1 rep max
  • Set 3: 5 reps @ 55% of 1 rep max

Exercise #2: DB Pullover with Chin Ups (weighted) superset (3 sets)

  • Set 1: 5 DB Pullovers + 3 Chin Ups
  • Set 2: 5 DB Pullovers + 3 Chin Ups
  • Set 3: 5 DB Pullovers + 3 Chin Ups

Exercise #3: Bench Press (5 sets)

  • Set 1: 5 reps @ 70% of 1 rep max
  • Set 2: 4 reps @ 80% of 1 rep max
  • Set 3: 3 reps @ 85% of 1 rep max
  • Set 4: 2 reps @ 90% of 1 rep max
  • Set 5: 1 reps @ 95% of 1 rep max

Exercise #4: DB Laterals (5 sets)

  • Set 1: 5 reps
  • Set 2: 5 reps
  • Set 3: 10 reps
  • Set 4: 10 reps
  • Set 5: 15 reps

Cardio: Weighted Push Ups (3 sets)

  • Set 1: 10 reps
  • Set 2: 10 reps
